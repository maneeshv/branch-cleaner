# Code of Conduct

## Our Pledge

We pledge to make participation in this project a respectful and harassment-free
experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, but take responsibility for impact.
- Keep technical disagreement focused on the work.
- Avoid personal attacks, harassment, and discriminatory language.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or contributors
that violate this code of conduct. Serious or repeated violations may result in
a permanent ban from project spaces.

Report concerns privately to the project maintainer.
